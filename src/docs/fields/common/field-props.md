> 通用的 fieldProps

| 参数         | 说明                         | 类型               | 默认值 |
| ------------ | ---------------------------- | ------------------ | ------ |
| preRender    | 在表单项之前渲染节点         | `ReactNode`        | -      |
| preGap       | 表单项与`preRender`的间隔    | `number \| string` | 8      |
| preStyle     | 在表单项之前渲染节点的样式   | `CSSProperties`    | -      |
| suffixRender | 在表单项之后渲染节点         | `ReactNode`        | -      |
| suffixGap    | 表单项与`suffixRender`的间隔 | `number \| string` | 8      |
| suffixStyle  | 在表单项之后渲染节点的样式   | `CSSProperties`    | -      |
