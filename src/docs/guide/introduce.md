# Antd-MS

Antd-MS 是基于 antd 而开发的模板组件，根据马上云，devops 的实际业务场景提供更高级别的抽象，开箱即用。可以显著的提升项目开发效率，更加专注于业务。

- <a href="/components/ms-layout" >MsLayout</a> 解决布局的问题，提供开箱即用的菜单和面包屑功能
- <a href="/components/ms-table">MsTable</a> 表格模板组件，抽象网络请求和表格格式化
- <a href="/components/ms-form">MsForm</a> 表单模板组件，预设常见布局和行为
- <a href="/components/ms-descriptions">MsDescriptions</a> 列表模板组件，可复用 MsForm 配置

在使用之前可以查看一下典型的 Demo 来判断组件是否适合你的业务。Antd-MS 是根据马上云，devops 业务场景提炼的, 预设了相当多的样式和行为。这些行为和样式更改起来会比较困难，如果你的业务需要丰富的自定义建议直接使用 Ant Design。

## 按需加载

`Antd-MS` 默认支持基于 ES modules 的 tree shaking。

## TypeScript

`Antd-MS` 使用 TypeScript 进行书写并提供了完整的定义文件，无需引入 `@types`。
