# 全局样式

## 1.色彩

色彩对产品的视觉传达和多维度信息的承载有着很重要的影响，好的产品在色彩配置上应当清晰，准确的传达有效信息和品牌感。

主色采用马上品牌色，与公司产品调性形成统一，同时定义中性色、功能色、图标色。

<img src="/images/design/color.png">

## 2.字体

优先使用各个系统中默认的无衬线字体。

定义常规使用字号为 14px，如页面内容过多，最小主字号可定义到 12px。

## 7.栅格

栅格可以有效的保证页面的一致性、逻辑性、加强团队协作和统一。

本系统采用 24 栅格设计，适用于业务信息量较大，信息分组较多；更加灵活，更适合内容比较多样的复杂场景设计。
